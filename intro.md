# 8x Student Preview

Welcome to **Data 8X: Foundations of Data Science** course materials.

This book contains lecture and lab notebooks for all three parts of the program:

- **Part 1 - Computational Thinking with Python:** programming in Python, working with data, visualization, and simulation
- **Part 2 - Inferential Thinking through Simulations:** hypothesis tests, confidence intervals, resampling, and drawing conclusions from data
- **Part 3 - Machine Learning and Predictions:** regression, classification, and building predictive models

Use the sidebar to navigate to a lecture or lab. To run notebooks interactively, open them in JupyterLab or use the "Run in Binder" link above.

For course overviews and edX enrollment, see the [Data 8X program page](https://data8.org/data8x/).
